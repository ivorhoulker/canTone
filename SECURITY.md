# Security Policy

## Supported Versions
We currently only provide security updates for the latest major release of the application. Please ensure you are running the latest version from our [Releases page](../releases/latest) before submitting a report.

## Reporting a Vulnerability

We take the security of our application seriously. 

If you discover a security vulnerability, **please do not open a public issue.**

Instead, please report it privately by emailing:
**[ivor@rooftopproductions.hk]**

Please include:
* A description of the vulnerability.
* The OS and application version you are using.
* Detailed steps to reproduce the exploit.

We will acknowledge your report within 48 hours and work with you to patch the vulnerability before public disclosure. Thank you for practicing responsible disclosure!
